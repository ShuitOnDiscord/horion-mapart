Due to limitations minecraft put on NPCs, the commands have to be split up into multiple NPCs. To combat this, the generator puts all the npcs into one block and puts a command in each npcs to open the next NPCs dialogue
Status: IDLE
Usage
  • Use MapArtcraft to turn your image into a nbt
  • From the left, select the blocks you want to use, if you're not sure, just use the Everything preset
  • Adjust the options on the right (mess around with dithering and brightness to make your image look best)
  • Hit the Download NBT button, and import the downloaded .nbt file here
  • The file will automatically be converted and you can copy it from the textbox or download it as a text file
